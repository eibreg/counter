# Keyboard Counter

A simple yet interactive web application that increments a counter every time you press a key on your keyboard. Perfect for quick counting tasks or just for fun!

## Features

- **Key Press Detection**: Increments the counter with any key press
- **Visual Feedback**: Animated counter with pulse effect on increment
- **Session Tracking**: Tracks your counting session with real-time statistics
- **Sound Effects**: Optional click sound on each key press
- **Auto-clicker**: Automated counting with one click
- **Responsive Design**: Works on desktop and mobile devices
- **Persistent Storage**: Saves your count between sessions using localStorage

## How to Use

1. Open `index.html` in your web browser
2. Press any key on your keyboard to increment the counter
3. Use the control buttons to:
   - Reset the counter
   - Toggle auto-clicker mode (also activated with 'A' key)
   - Enable/disable sound effects
4. View your counting statistics in the dashboard

## Live Demo

You can try the application directly by opening the `index.html` file in any modern web browser. No server required!
